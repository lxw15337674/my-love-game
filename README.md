# My LÖVE Game

A tiny playable LÖVE 11.x starter project.

## Run

```bash
love .
```

## Controls

- `Enter` start / restart
- `WASD` or arrow keys move
- `Esc` quit

## Structure

- `main.lua` - game loop and starter gameplay
- `conf.lua` - LÖVE config
- `systems/` - game systems go here
- `ui/` - UI code goes here
- `data/` - game data/config goes here
- `assets/` - images, fonts, sounds
- `docs/` - notes or web docs
