# Robot War Prototype Boss 阵容设计

> 目标：Boss 战不再重复同一个模板。Boss 波从 20 个 Boss 池随机抽取；Boss 只保留基础模板数值，实际强度由所在关卡的 wave 缩放、章节 Boss 压力、词缀和危险等级动态平衡。

## 设计参考口径

参考大型游戏的“机制组织方式”，不照抄具体 IP、Boss 名称或技能包装。

- 参考《无主之地》：元素克制、护盾 / 护甲 / 血肉差异、弱点窗口、构筑检验、Boss 掉落感。
- 参考《魔兽世界》：阶段转换、清晰地面预警、add 小怪、软狂暴、机制教学递进、读条/前摇让玩家有反应时间。
- 本项目落地原则：每个 Boss 只负责 1 个主机制 + 1 个副压力，避免第一版变成机制垃圾场。
- 所有危险必须有可读预警：颜色、文字、弹幕形状、地面圈至少命中两种提示。

## 随机与数值规则

- 每个关底 Boss 波开始时，从 20 个 Boss 模板中随机抽取 1 个并锁定本波。
- 避免连续两次抽到同一个 Boss，降低重复感。
- Boss 基础模板只定义相对定位：血量型、护盾型、速度型、伤害型、机制型。
- 入场时继续走统一缩放链路：`wavePowerScale(Game.wave) × currentAffixBonuses() × survivalEnemyCurve() × dangerScale`。
- 第一大关仍保留教学保护：首个 Boss 波降低血盾和护盾回复，避免新机制叠加数值墙。
- 随机 Boss 不绑定固定大关；同一个 Boss 放到 wave 3 和 wave 21，靠动态缩放、章节词缀、支援敌人和刷怪压力区分难度。
- 前两章是 Boss 机制教学段：第一 Boss 地面区只做视觉教学、禁普通刷怪和 Boss 召唤；第二 Boss 降低伤害/区域/支援压力；第三章后逐步恢复完整 Boss 关压力。

## Boss 列表

### 1. 裂心机核

- 定位：弹幕 / 封锁教学。
- 参考味道：WoW 式阶段教学 + 无主之地式弱点窗口。
- 核心机制：扇形弹幕、裂隙封锁区、核心暴露弱点窗口。
- 阶段：
  1. 校准射击：低密度扇形弹幕，教玩家看 Boss 朝向。
  2. 裂隙封锁：地面紫色封锁区切走位。
  3. 核心暴露：短弱点窗口 + 高频齐射。
- 压力点：让玩家学习“看 Boss 状态 + 处理地面危险”。
- 克制提示：持续输出、爆发窗口、位移保命。
- 风险边界：不能让封锁区完全堵死逃生路线。

### 2. 赤炉执刑者

- 定位：燃烧区域 / 近身压迫。
- 参考味道：无主之地火属性克制血肉 + WoW 火圈躲避。
- 核心机制：熔炉线、燃烧区、过热阶段高频弹幕。
- 阶段：
  1. 热炉点火：小范围燃烧区逼玩家移动。
  2. 熔线横扫：燃烧区出现在玩家移动路径前方。
  3. 过热审判：扇形弹幕与燃烧区交替出现。
- 压力点：限制站桩，惩罚长期停在同一区域。
- 克制提示：移速、护盾回复、远程覆盖、冰霜减速。
- 风险边界：火圈伤害不能和敌群碰撞同时秒杀玩家。

### 3. 铁幕壁垒

- 定位：高护盾 / 召唤卫士。
- 参考味道：无主之地护盾条克制 + WoW add 管理。
- 核心机制：高护盾、护盾卫士支援、破盾后反扑。
- 阶段：
  1. 护盾校准：低速高盾，给玩家识别破盾需求。
  2. 壁垒增殖：周期召唤护盾卫士。
  3. 破盾反扑：破盾后弹幕变密，但本体血量压力较低。
- 压力点：检验破盾能力和清小怪效率。
- 克制提示：电弧、破盾、腐蚀易伤。
- 风险边界：护盾回复不能盖过普通玩家的合理输出。

### 4. 蜂巢母机

- 定位：召唤 / 弹幕密度。
- 参考味道：WoW 小怪阶段 + 无主之地蜂群武器爽感反向压迫。
- 核心机制：周期召唤裂片/漂移噪声，多弹幕扇形压场。
- 阶段：
  1. 蜂群试探：少量小怪，提示清场需求。
  2. 母巢分裂：召唤频率提高，小怪从多方向入场。
  3. 集群暴走：弹幕更宽，小怪成为主要压力。
- 压力点：若清场不足，屏幕会被小怪挤满。
- 克制提示：AOE、多弹、穿透、召唤反制。
- 风险边界：小怪数量要受 cap 限制，不能用纯数量淹死性能和视野。

### 5. 冷井裁决体

- 定位：减速区 / 节奏冻结。
- 参考味道：WoW 冰圈 + 无主之地冰冻控场。
- 核心机制：霜环区域、低温弹幕、后期冻结节奏压迫。
- 阶段：
  1. 低温锁定：低速冰弹，让玩家学会避开冷色危险。
  2. 霜环切割：地面减速区分割路线。
  3. 冻结裁决：减速区更大，弹幕节奏更快。
- 压力点：玩家移动路线被拖慢，容错下降。
- 克制提示：射程、弹速、护盾保底、火焰爆发。
- 风险边界：减速不应让玩家失去控制感。

### 6. 蚀刻孢群

- 定位：腐蚀区 / 区域逼位。
- 参考味道：无主之地腐蚀打护甲 + WoW 毒云换位。
- 核心机制：腐蚀云、持续压位、衰变爆发。
- 阶段：
  1. 腐蚀孢子：小范围毒云，提醒玩家别站桩。
  2. 毒圈扩散：毒云更频繁，压缩安全区域。
  3. 衰变爆发：弹幕和毒云错峰出现，逼玩家管理路线。
- 压力点：长时间被迫换位，拖慢输出节奏。
- 克制提示：高爆发、吸血、区域清理、远程自动武器。
- 风险边界：毒云持续时间不能过长，否则安全区会被永久污染。

### 7. 黑箱坍缩核

- 定位：牵引 / 密集惩罚。
- 参考味道：WoW 引力机制 + 无主之地黑洞/奇点手雷。
- 核心机制：周期牵引玩家、坍缩点、虚空弹幕。
- 阶段：
  1. 引力锁定：轻微牵引，提醒玩家修正站位。
  2. 黑箱牵引：牵引叠加地面坍缩点。
  3. 坍缩奇点：牵引与密集弹幕错峰出现。
- 压力点：破坏玩家站位，和敌群叠加后风险高。
- 克制提示：位移、远程、护盾保命、虚空/电弧清场。
- 风险边界：牵引不能直接把玩家拖进必死区。

### 8. 白噪狙击塔

- 定位：高伤狙击 / 走位校验。
- 参考味道：WoW 锁线技能 + 无主之地狙击敌人的高惩罚低频攻击。
- 核心机制：少量高速高伤锁线，后期交叉狙击。
- 阶段：
  1. 测距锁线：单线高伤，强前摇。
  2. 交叉狙击：双线/三线压走位。
  3. 白噪齐射：锁线间隔缩短，但保留可躲窗口。
- 压力点：弹幕数量不多，但命中惩罚高。
- 克制提示：速度、预判、护盾上限、冲刺无敌。
- 风险边界：锁线必须有足够前摇，不能变成不可读瞬发。

### 9. 天灾反应堆

- 定位：爆炸环 / 高压终盘。
- 参考味道：WoW 软狂暴 + 无主之地爆炸属性压迫。
- 核心机制：环形泄压、爆炸区、熔毁阶段高频压制。
- 阶段：
  1. 反应堆升温：环形弹幕建立节奏。
  2. 灾变泄压：环形弹幕 + 地面爆炸区。
  3. 核心熔毁：软狂暴，高频但短窗口循环。
- 压力点：后期压力 Boss，要求玩家同时处理弹幕和地面危险。
- 克制提示：持续输出、吸血、护盾回复、爆发窗口。
- 风险边界：软狂暴是压力，不是强制秒杀倒计时。

### 10. 重启终端

- 定位：混合机制 / 最终综合考。
- 参考味道：WoW 最终 Boss 多阶段复盘 + 无主之地多元素切换。
- 核心机制：协议重放、多态切换、归零点。
- 阶段：
  1. 协议重放：复用基础扇形弹幕，检查基本走位。
  2. 多态切换：切到燃烧/封锁压力。
  3. 重启归零：混入虚空坍缩点，检查综合构筑。
- 压力点：混合前面 Boss 的机制语言，检验构筑完整度。
- 克制提示：均衡构筑、爆发窗口、位移容错。
- 风险边界：混合机制每次只能叠 1-2 个，不做同时全开。


### 11. 电磁审判庭

- 定位：电弧链 / 破盾压力。
- 参考味道：无主之地电击破盾 + WoW 连线分摊/散开。
- 核心机制：雷场、电弧链式裁决、过载阶段高频电弹。
- 压力点：护盾构筑会被持续检验，密集敌群会放大电弧压力。
- 克制提示：腐蚀易伤、爆发破盾、保持移动。
- 风险边界：链式伤害不得递归触发异常，避免 stack overflow。

### 12. 量子镜像体

- 定位：残像 / 目标切换。
- 参考味道：WoW 分身 add + 无主之地高速机动敌。
- 核心机制：残像召唤、错位弹幕、镜像分裂。
- 压力点：干扰自动锁敌和玩家判断。
- 克制提示：AOE、穿透、自动追踪。
- 风险边界：残像只能用普通小怪表达，不制造真假 Boss 混淆。

### 13. 回收圣棺

- 定位：恢复 / 持久战。
- 参考味道：WoW 治疗窗口打断感 + 无主之地护盾回复压力。
- 核心机制：周期自我修复、护盾回补、宝藏信标诱饵。
- 压力点：输出不足会被拖入长战。
- 克制提示：爆发、腐蚀、暴击窗口。
- 风险边界：回血有上限，不能抵消普通玩家合理输出。

### 14. 地雷织网机

- 定位：地雷区 / 路线规划。
- 参考味道：WoW 地面雷区 + 无主之地爆炸物。
- 核心机制：布雷、网格封锁、连锁爆破。
- 压力点：让玩家提前规划移动路线。
- 克制提示：远程输出、移速、护盾容错。
- 风险边界：地雷不能铺满全场，必须保留安全通道。

### 15. 碎星决斗者

- 定位：冲刺 / 单点高压。
- 参考味道：WoW 点名冲锋 + 无主之地近战精英。
- 核心机制：锁定挑战、突刺连段、处刑星轨。
- 压力点：少量高威胁攻击，检验冲刺和预判。
- 克制提示：冲刺无敌、减速、远程风筝。
- 风险边界：冲刺必须有文字/方向预警，不能瞬发撞死。

### 16. 棱镜分光仪

- 定位：多元素 / 读色反应。
- 参考味道：无主之地多元素抗性 + WoW 颜色机制。
- 核心机制：分光弹幕、元素折射、全谱压制。
- 压力点：玩家需要识别颜色和区域危险。
- 克制提示：均衡构筑、通用动能、护盾保底。
- 风险边界：颜色不能成为唯一提示，必须配文字/形状。

### 17. 深井压缩者

- 定位：重力井 / 站位破坏。
- 参考味道：WoW 引力井 + 无主之地奇点手雷。
- 核心机制：重力牵引、压力折叠、深井塌缩。
- 压力点：把玩家从舒适输出位拉走。
- 克制提示：位移、远程、召唤/自动武器。
- 风险边界：牵引不能直接拖入必死区。

### 18. 血肉缝合塔

- 定位：厚血 / 小怪献祭。
- 参考味道：WoW 缝合怪 add + 无主之地血肉 Boss。
- 核心机制：血肉增殖、缝合护卫、畸变狂潮。
- 压力点：厚血本体叠加小怪潮，检验续航和清场。
- 克制提示：火焰、腐蚀、吸血、AOE。
- 风险边界：小怪召唤频率要受 cap 限制。

### 19. 零度列车

- 定位：横冲 / 轨道预警。
- 参考味道：WoW 火车/冲锋线机制 + 冰霜减速。
- 核心机制：寒潮轨道、冲撞、终点急冻。
- 压力点：逼玩家横向躲轨道，同时处理冰弹。
- 克制提示：移速、冲刺无敌、火焰爆发。
- 风险边界：轨道区必须有足够前摇。

### 20. 终焉播报机

- 定位：混合广播 / 节奏干扰。
- 参考味道：WoW 语音播报机制 + 无主之地混合元素战。
- 核心机制：死亡播报、协议串扰、终焉倒放。
- 压力点：用不同颜色/召唤/区域组合打乱节奏。
- 克制提示：均衡构筑、优先清 add、保留位移。
- 风险边界：混合机制同时最多叠两类，避免视觉污染。

- 已验证：`luac -p main.lua` 和 `git diff --check` 通过；`balanced` 直跑 wave 3 / 6 / 9 / 12 均可击破 Boss；新增强制测试入口 `LOVE_AUTOPLAY_BOSS_ID` 可逐个点名回归。
